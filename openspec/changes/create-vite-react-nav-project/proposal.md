## Why

Criar um novo projeto React moderno com Vite como build tool, utilizando React Navigation para gerenciamento de rotas cliente-side, TypeScript para tipagem forte, Tailwind CSS para estilização e Shadcn UI para componentes de interface consistentes.

## What Changes

- **New**: Projeto React com Vite setup completo no diretório atual
- **New**: Navegação baseada em componentes com React Navigation v6+
- **New**: Configuração de Tailwind CSS com directives @tailwind base, components e utilities
- **New**: Integração com Shadcn UI para componentes de interface prontos
- **New**: Página inicial com conteúdo básico "Olá Mundo"
- **New**: Estrutura de pastas organizada para aplicação React SPA

## Capabilities

### New Capabilities

- vite-react-project: Configuração completa do projeto React com Vite, TypeScript e ESLint
- react-navigation-routes: Implementação de navegação com React Navigation, stacks de telas e provedor de contexto de navegação
- tailwind-css-config: Configuração inicial do Tailwind com content paths definidos para componentes Shadcn
- shadcn-ui-setup: Instalação e configuração do Shadcn UI com componentes básicos (Button, Card, etc.)
- basic-home-page: Tela inicial com componente simples de "Olá Mundo" e layout básico
