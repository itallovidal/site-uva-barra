## Why

The project needs a modern, fast, and type-safe frontend foundation. React with Vite provides fast development experience and optimized production builds. React Navigation enables declarative, type-safe routing. TypeScript ensures code quality and developer productivity.

## What Changes

- Scaffold a new React + Vite project with TypeScript
- Configure React Navigation for client-side routing
- Set up project structure with TypeScript strict mode
- Add base configuration for linting, formatting, and development tooling

## Capabilities

### New Capabilities
- `project-scaffold`: Vite + React + TypeScript project setup with build and dev scripts
- `routing`: React Navigation configuration with typed routes and navigation structure
- `project-structure`: Standardized directory layout, base components, and TypeScript configuration

### Modified Capabilities

## Impact

- Creates new project files in the current directory
- Adds dependencies: react, react-dom, vite, @vitejs/plugin-react, react-router-dom (or react-navigation for web), typescript, @types/react, @types/react-dom
- Establishes development workflow with Vite dev server and build pipeline
